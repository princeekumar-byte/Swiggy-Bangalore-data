Overview
This project uses Python to analyze restaurant data from Swiggy in Bangalore. It cleans raw data and creates visualizations to show the most popular food hubs in the city.

Features
Data Cleaning: Removes unnecessary information like URLs and offers while fixing missing values.

Cuisine Analysis: Calculates the variety of cuisines offered by each restaurant.

Area Insights: Groups data by neighborhood to find average costs and ratings.

Visualization: Includes a bar chart showing the Top 10 busiest areas for food.

Files
Swiggy Bangalore.csv: The original raw dataset.

Swiggy_Bangalore_Cleaned.csv: The processed and informative data.

Analysis_Script.py: Python code for data processing and plotting.

How to Run
Install pandas and matplotlib: pip install pandas matplotlib.

Run the script to generate the cleaned CSV and the analysis plot.
